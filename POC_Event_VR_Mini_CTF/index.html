Welcome to VR GAME HACKING CTF CHALLEGE

The Rules

#Challege List#

  • #1 Testing Your Memory Power (10 minutes)
  • #2 Spooky Calling (10 minutes)
  • #3 INFINITE Boss (15 minutes)
  • #4 Sub mission ( There is no fixed time. If you want to solve it, tell the staff.)

For information about each game, go to About menu.

#How to Solve#

  • Main challenge
  • We have the basic tools to solve the problem.
    Tools required are downloadable.
  • Scores are reduced by one point every 10 seconds from the start of problem solving.
  • If you can't solve the problem in time, your score is zero.
  • If you understand everything, please inform administrator that you are ready.

  • Sub Mission [[New challenge]]
  • There are 25 words in total, so if you memorize the words here,
    you will get 1 point for each number
  • If you find the word, please call the event staff! I will confirm.

#문제 풀이 방법#

  • Main challenge
  • 문제풀이에 필요한 기본적인 툴들만 갖춰져있는 상태입니다.
    추가로 필요한 툴들은 다운로드하여 사용하시면 됩니다.
  • 점수는 문제 풀이 시작 후, 10초가 지날 때 마다 1점씩 감소됩니다.
  • 만약 주어진 시간 내에 문제를 풀지 못하면 0점 처리됩니다.
  • 모든 내용을 이해했다면, 운영진에게 준비가 되었음을 알려주세요!

  • Sub mission[[New challenge]]
  • 총 25단어가 있습니다, 단어를 기억하신다면 한단어당 1점을 얻게됩니다.
  • 단어를 찾으신다면 이벤트 운영진을 불러주세요! 확인해드리겠습니다.

Have fun! :)
